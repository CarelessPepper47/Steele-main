2. Updating Variables

let score = 5;

score += 5 // 10

score // 10

score -= 2 // 8

score // 8

There is shorter version:

score-- // 7 -- it's decrementing by 1 using this way

score // 7

score++ // 8

score // 8

Difference between i++ and ++i:

i++ : first returns the original value AND THEN it increases it by 1

++i : returns already already increased value

let i = 0;

i++ // 0; i = 1

i // 1

++i // 2; i = 2

i // 2
