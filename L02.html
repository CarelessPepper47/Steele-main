-   < less than

-   >= greater than or equal to

-   <= less than or equal to

-   == equal

-   != not equal

-   === strict equality

1. Equality(==) vs Strict Equality(===)

1 == 1 // true

1 == "1" // true > it doesnt matter if the type is different, if after transforming the argument, they are equal after all

1 === 1 // true

1 === "1" // false > even tho both are 1, one is number (typeOf) and the second is string

3. Additional + Logical Operators (&& ; || ; !)

- && // both sides have to be true for the whole operation to be true

1 <= 4 && "a" === "a" // true

4 > 16 && 4 === 4 // false

abc.lenght === 3 && 1+1 === 4 // false

better example in L02.js :58

- || // only one side has to be true for the whole thing to be true

1 !== 1 || 10 === 10 // true

10/2 === 5 || null // true

0 || undefined // false
