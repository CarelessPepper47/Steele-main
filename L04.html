1. Introduction

Objects are a collection of properties; which contain a pair made of a key-value.

let uzytkownicy = [

{imie:"Morel",

wiek: 26,

but: 40,

log: true},

{imie: "Olek",

wiek: 30,

but: 35,

log: true},

{imie: "Kubeb",

wiek: 17,

but: 39,

log:false}

]

Accessing data inside is used by keys and NOT by index

uzytkownicy["imie"] // undefined

uzytkownicy[1]["imie"] // "Olek"

uzytkownicy[2]["but"] // "39"

uzytkownicy[1].imie // "Olek" > also works

Every key is converted into "string"

uzytkownicy[1].imie = "Geh" // "Geh" > "Olek"
