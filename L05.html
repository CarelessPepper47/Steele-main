5. Iterating over Objects

Arrays, even tho they are objects, are iterated by loop command for... OF. Iterating over Objects is different : for (let key IN object)

const midterms = {

Adam: 98,

Lol: 86,

Git: 69,

Goge: 24

}

for (let person in midterms) { // let person IN midterms, NOT OF

console.log(`${person}`) // Adam, Lol, Git, Goge

console.log(`${person} scored: ${midterms[person]}`) // Adam scored: 98, Lol scored: 86, Git scored: 69, Goge scored: 24

}
